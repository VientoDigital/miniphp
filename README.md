# Install


```bash
composer install
```

```bash
npm install
```

## Development
### Tailwind CSS

For development use `--watch`
```bash
npx tailwindcss -i ./public/css/app.css -o ./public/css/build/app.css --watch
```

### Alpine Js & Javascript

```bash
npm run dev
```

## Production

```bash
npm run build
```
